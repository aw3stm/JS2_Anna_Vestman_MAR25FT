# Rantr - Social Media Client

A social media web app built as a course assignment for Noroff. It interacts with Noroff Social API, allowing users to register, log in, interact with others and manage their own profile.

## Live Demo [View the live application](https://69f67ecc2bb2c000083ec194--js2annavestmanmar25ft.netlify.app/)

## Features

- **Authentication** Register a new account (JWT based).
- **Feed** Latest posts from all users
- **Single post view** Click on a post to view its full content, with comments and reactions.
- **CRUD** Create, view, edit and delete your own posts.
- **Interactions** React and comment posts.
- **Profiles** View your own and other user's profiles.
- **Follow and Unfollow** Follow or Unfollow users.
- **Search** Search user's posts or usernames.

## Built with

- HTML5 and CSS3
- Vanilla TypeScript / JavaScript (ES6 Modules)
- Vite (Build tool)
- [Noroff V2 API](https://docs.noroff.dev/)
